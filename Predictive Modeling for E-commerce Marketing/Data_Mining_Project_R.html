Brand Frequency Encoding in Training Data and Validation Data¶

We have 240 brand categories which is not feasable to run in the model, have applied frequency encoding to convert categorical variables into numeric representations based on the frequency of their occurrence in the dataset. It involves replacing the original categorical values with their corresponding frequencies, which represent how often each category appears in the dataset.

In [49]:
freq_encode <- function(x) {
 freq <- table(x)/length(x)
 return(freq[x])
}
training.df$brand <- freq_encode(training.df$brand)

training.df$brand_freq <- brand_freq[as.character(training.df$brand)]
